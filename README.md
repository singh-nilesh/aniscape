### Installation  
1. You should have latest `node` module instated, `sudo apt install nodejs` 
1. Clone Repository
   ```bash
   git clone https://github.com/singh-nilesh/aniscape.git
   ```

1. Install dependencies
   ```bash
   npm install
   ```

2. Start the app
   ```bash
    npx expo start
   ```
